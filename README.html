How to deploy

This site reads your Google Sheet directly in the browser (no server). It’s prefilled with your Sheet ID (1ma4hOfTkjOG36cxn-pZnJyL8o5jeIxUn) and GID (2062709776). You can change them from the top bar on the page.

1) Prepare the Google Sheet

  1. Make the sheet tab (gid 2062709776) publicly viewable: Share → Anyone with the link (Viewer) or File → Publish to the web.
  2. Ensure columns for University and Program exist (names can vary; you can map columns on the page).

2) Deploy to GitHub Pages

  1. Create a public repo on GitHub.
  2. Upload index.html and README.html.
  3. Settings → Pages → Source: Deploy from a branch → Branch: main → Folder: / (root).
  4. Wait ~1 minute. Your site appears at https://<username>.github.io/<repo>.

3) Or deploy to Netlify

  1. Sign in at netlify.com.
  2. Drag & drop the folder to create a new site (or link your GitHub repo).
  3. Netlify gives you a URL like https://xxxxx.netlify.app.

4) Using the page

  • Pick a program from the dropdown or search by keyword.
  • Matching universities will appear instantly; click Export CSV to download the list.
  • If auto-detection of columns is off, use the “Column Mapping” box to set University / Program fields.
